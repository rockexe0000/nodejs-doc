## Setup

### install nodejs

```shell
nvm install 10.14.1
nvm use 10.14.1
```

> nvm use 版本報錯，出現exit status 1後面跟一堆亂碼<br>
> 此時是因為無權限，以管理員身份打開，再次使用nvm use，問題解決



### version verification

```shell
# nvm version
1.1.8

# node -v
v10.14.1


```
